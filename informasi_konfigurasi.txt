# INFORMASI APLIKASI DAN KONFIGURASI (TUGAS PBO)

## 1. Deskripsi Aplikasi
**Dial Ajar** adalah sebuah platform _Learning Management System (LMS)_ modern yang memfasilitasi interaksi belajar mengajar antara Dosen dan Mahasiswa. Aplikasi ini menyediakan sistem manajemen mata kuliah, materi ajar, evaluasi (tugas & kuis otomatis), hingga fitur AI terintegrasi menggunakan Google Gemini.

## 2. Framework & Teknologi Utama
- **Frontend**: React (Vite) + Tailwind CSS
- **Backend**: Java Spring Boot
- **Database**: MySQL

## 5. Cara Menjalankan Aplikasi

### [Backend - Spring Boot]
1. Buka aplikasi Terminal / Command Prompt.
2. Arahkan ke dalam folder `backend` (`cd backend`).
3. Pastikan **JDK 21** sudah terinstal.
4. Jalankan perintah berikut untuk menyalakan server:
   `.\mvnw spring-boot:run`  (Untuk Windows)
   `./mvnw spring-boot:run`  (Untuk Mac/Linux)
5. Tunggu hingga backend sukses berjalan di `http://localhost:8080`.

### [Frontend - React/Vite]
1. Buka Terminal baru, arahkan ke dalam folder `frontend` (`cd frontend`).
2. Instal semua dependensi Node.js dengan perintah:
   `npm install`
3. Pastikan konfigurasi API Key (langkah ke-4 di atas) sudah Anda lakukan.
4. Jalankan server antarmuka dengan perintah:
   `npm run dev`
5. Aplikasi dapat diakses melalui browser pada alamat `http://localhost:5173`.
